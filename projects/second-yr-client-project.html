NHS Wales Recruitment App

This project was key to my development as a software developer. I utilised this project to learn more about various aspects of software development like DevOps, Cyber Security and Performance & Scalability. It is a recruitment application, meant for internal use by NHS Wales. The application allows users to create an account, login, and review applications to job openings in the organisation. It has a variety of features such as a dynamic dashboard and application profile for each item in the database. The ability to email candidates, admin page, statistics page, as well as much more. The application backend was developed using Springboot, a widely used Java framework for developing web applications and taught me how to make modern web applications easily. Additionally, I developed the project in a team of 4, which taught me how to work in a team using the Agile methodology. This was critical to the success of this project as it made the development process more efficient and effective. The agile framework is used widely in the industry and is a key skill to have as a software developer. If you contact me, I can provide you with a demo of the application and go into further detail.
